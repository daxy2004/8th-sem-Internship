## 🌍 Project 1 — GlobalXcelerate Corporate Website

**GlobalXcelerate** is an international education and career acceleration company with offices in **Dubai (UAE)**, **Singapore**, and **Bengaluru (India)**. The company runs global immersion programs that take students across 12+ countries including UAE, Singapore, Taiwan, England, Scotland, Japan, Australia, Vietnam, Malaysia, Indonesia, South Korea, and the USA.

### What We Built

A fully responsive, multi-page corporate website that serves as the digital face of GlobalXcelerate. The website:

- Showcases all **services** offered — Cultural Immersion, Global Alliances, Career Development, Language Training, Placement, AI Tools, and Study Abroad programs
- Provides dedicated **country cohort pages** for each of the 12+ destination countries, complete with program details and highlights
- Features a **photo gallery** capturing moments from past cohort programs
- Includes a **contact page** for prospective students and partners to reach out
- Is built as a **Single Page Application (SPA)** using React 19 with client-side routing via React Router v7, bundled with Vite for fast performance

### Key Highlights

- 🗺️ **12 Country Cohort Pages** — each with destination-specific content and imagery
- 🛠️ **7 Service Detail Pages** — deep dives into each program offering
- 📱 **Fully Responsive** — optimized for mobile, tablet, and desktop
- ⚡ **Fast & Modern** — Vite-powered build, TypeScript for type safety
- 🎨 **Custom Design System** — hand-crafted CSS with animations, glassmorphism effects, and a consistent brand identity
